Our Products and Partners

TouchOS

TouchOS is an operating system which is built for even the oldest of machines. It's extensive compatibility allows it to run applications which were orginally built for older machines that wouldn't be compatible with newer computers which don't run TouchOS.

Learn more about TouchOS here.

TouchOS CE

TouchOS CE is the compact edition of TouchOS, optimized for smaller devices and embedded systems. It retains the key features of TouchOS while being resource-efficient for a wide range of applications.

Learn more about TouchOS CE here.

NetFruit Technologies

NetFruit Technologies is one of our main partners and is able to create the best cutting-edge applications for user convenience and developers. They are also helping us with the development of NetOS. Visit their website for their latest updates and projects.

DeskTop

DeskTop is a completely free and more simple collection of business applications with a text editor and spreadsheet maker.

Learn more about DeskTop here.

DeskTop Home Edition

DeskTop Home Edition is a user-friendly version of DeskTop, designed for home users. It provides more home oriented applications which regular DeskTop doesn't have.

Learn more about DeskTop Home Edition here.

Spider

Spider is our flagship collaborated operating system which is being made by us, NetFruit Technologies and TDI, designed to meet the demands of more modern computers. It offers a much more sleek design and is marketed towards everyone.
